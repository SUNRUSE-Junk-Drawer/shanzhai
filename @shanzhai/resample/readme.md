Adjust the sample rate of floating-point audio.

## Usage

```typescript
import { resample } from "@shanzhai/resample";

const inputSamples = [-1, -0.5, 0, 0.5, 1];

const outputSamples = resample(inputSamples, 44100, 22050);
```
